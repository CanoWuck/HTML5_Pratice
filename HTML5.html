HTML 清單

項目編號(Ordered List)

HTML 清單提供數字順序的項目編號，ol 標籤可以建立項目編號

  1. 項目 1
  2. 項目 2

屬性:

  • start 屬性: 指定項目編號的開始
  • type 屬性: 指定項目編號是數字、英文等，EX:1、A、Ⅰ、i
  • reversed 屬性: HTML5 新增屬性，指定項目編號是反向由大至小

項目符號(Unordered List)

無編號的項目符號，即在項目前是小圓形、正方形等符號

  • item 1
  • item 2

定義清單(Definition List)

HTML 5 定義清單是一個任何名稱和值成對群組的結合清單，EX:詞彙說明的每一個項目是定義和說明

< a href="http:www.flag.com.tw">< h3>旗標出版社< /h3>< /a>

超連結 標籤的屬性:

  • href 屬性: 指定超連結連接的目的地，其值可以是想對URL網址/同網站的檔案名稱，EX:default.html 或 http:www.flag.com.tw
  • hreflag屬性:指定連接HTML網頁的語言，EX: en、zh
  • media屬性:HTML5 新屬性，可指定哪一種媒體或裝置可以最佳化處理連接的網頁文件
  • rel屬性:指定目前網頁和連接網頁之前的關係，只有href 屬性存在時才能指定，EX:值alternate是替代文件; bookmark是用來作為書籤
  • ★target屬性:指定超連結如何開啟目的地的HTML 網頁
  • type屬性:指定連接HTML 網頁的MIME 狀態

target 屬性值:

  • _blank 屬性值: 在新視窗或新標籤開啟HTML
  • _self屬性值: 在原視窗或標籤開啟HTML
  • _top屬性值: 以全螢幕開啟HTML 網頁
  • _top屬性值: 在父框架開啟 HTML 網頁
  • iframeName屬性值:在指定名稱的 < iframe >框架開啟HTML 網頁

因為HTML5 不再支援框架頁，上述 -top、_parent 和 iframeName 屬性主要是使用在 < iframe >標籤

HTML5 的表格與容器標籤

表格唯一種資料編排方式，如果網頁內容分類，我們可以使用表個將資料分類和系統化處理，表格可以重新編排原本雜亂的資料來清楚呈現欲表現的資訊。

HTML 的 < div >和 < span >標籤是一種一班用途的結構標籤它如同是一個容器，可以用來放置其他HTML 元素

  • < table > 標籤: 建立表格，其他表格相關標籤都位在此標籤之中
  • < tr > 標籤: 定義表格的每一個表格列
  • < th > 標籤: 定義表格的標題列
  • < td >標籤: 定義表格列的每一個儲存格
  • < caption >標籤: 定義表格的標題文字，其位置是 < table >標籤的第1個子元素
  • < thead >標籤: 群組HTML表格的標題內容
  • < tbody >標籤: 群組HTML表格的本文內容
  • < tfoot >標籤: 群組HTML表格的註腳內容

HTML 5 表格只支援< table >標籤的 border 屬性，為邊框像素宽，EX:border: 1px solid black;。HTML5 的 < td >其說明如下表所示:

  • colspan 屬性: 指定表格列需要擴充幾個儲存格，及合併儲存格
  • rowspan 屬性: 指定表格欄需要擴充幾個儲存格，及合併儲存格
  • headers 屬性: 指定的屬性值是對應標題列的儲存格的 id 屬性值
