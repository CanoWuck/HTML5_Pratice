Javascript 內建函數

  • escape () 函數: 使用URL 編碼傳入的參數字串，可以傳回加碼後的字串如下所示: strURLcode = escape(strMsg);
  • unescape() 函數: 解碼參數的URL 編碼字串，可以傳回還原成編碼前的原始字串，如下所示:strOriginal = unescape(strURLcode);
字串的字元 說明
英文字母和數值 不進行編碼
空白、標點符號 轉換成%XX字串，XX為十六進為 ASCII碼，例如:空白字元轉換成 %20
中文字 轉換成 %uXXX字串，XXXX為十六進為統一字元碼，例如:「網」轉換成%u7DB2

建立 Javascript 函數

Javascript 函數是由function 關鍵字、函數名稱和程式區塊組成，如下所示:function writeString(){ document.write("歡迎使用 Javascript 函數!") }

擁有參數的 Javascript 函數:function writeNString(strMsg,intnumber){ }

Javascript 函數的傳回值: convert2F(c){ var f; f = (9.0*c)/5.0+32.0; return f; }
使用 函式:var f = convert2F(20);

資料型態方式說明
數值、字串和布林傳值參數傳遞和比較操作都是使用傳值方式
物件、陣列和函數傳址參數傳遞和比較操作都是使用傳址方式
字串物件傳址參數傳遞和比較操作都是使用傳址方式

var arra = new Array("a","b","c");//建立陣列arra

var arrb = new Array("a","b","c");//建立陣列arrb

document.write((arra==arrb));

document.write((arra.toString()==arrb.toString()));

上述程式碼建立陣列物件 arra 和 arrb 陣列元素相同，如果直接比較兩個陣列，因為是使用傳址。所以結果為 false。

如果需要檢查兩個陣列的元素內容是否相同，請使用 toString() 方法，此時為true
