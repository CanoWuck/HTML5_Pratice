範例: var strName,intBalance;

var strName = "基本功";

var intBalance = 1000;

var blnSex = true;


判斷 Javascript 的變數是否存在

Javascript 程式碼的變數需要使用 var 宣告或以指定敘述來隱藏宣告，對於一個變數，

程式碼如何知道他是否存在，所謂存在是指變數擁有值，而不是 undefined 資料型態

if (intBalance)

  document.write("intBalance 存在")

else

  document.write("intBalance 不存在")

上述 if 條件檢查變數值是否為 undefined, 如果 false 表示存在,否則變數不存在，問題是變數 intBalance 需要是已經宣告的變數,如果變數根本沒有宣告

此時的 if 條件會導致 javascript 執行錯誤,不過 在劉爛氣並不會顯示錯誤訊息,只是根本忽略此 if 條件。

資料型態的轉換函數

parseInt () 函數

將字串變數值開頭的數字轉換成整數，如果字串開頭不是數字，就回傳Nan(not a number)


parseFloat () 函數

將字串變數值開頭的浮點數字串轉換成浮點數。如果字串開頭不是數字，就回傳NaN


eval() 函數

將運算式的字串參數當作運算式，傳回運算式的計算結果

eval () 函數 值 說明
eval ("20+ 4 * 5") 40 算術運算式
eval ("intBalance = 1000") 1000 指定運算式
eval ("5 > 4) true 邏輯或比較運算式

typeof() 函數

typeof() 運算子可以取得變數的資料型態，也就是 string、number、boolean、undefined、object 等資料型態
